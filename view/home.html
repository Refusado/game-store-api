Game Store API

Uma API para realizar interações com o banco de dados de uma loja fictícia de jogos

Mantendo o projeto simples...

Não é necessário a utilização de headers para o consumo da API, é possível realizar todas as requisições HTTP disponíveis através de parâmetros passados na URL. (Há suporte para todas operações CRUD)

COMO FUNCIONA

Para realizar todas as operações é preciso enviar os parâmetros na URI da API, com isso você terá a URL final para conexão a depender da operação desejada.
