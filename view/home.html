Bem vindo à Game Store API

Este projeto simula a API de uma loja de jogos.
Para manter o projeto simples, não é necessário a utilização de headers para o consumo da API e apenas com a requisição da URL é possível utilizar todos os métodos disponíveis. (CRUD).
Nesta API é possível acessar, criar, atualizar e até deletar dados do banco.
